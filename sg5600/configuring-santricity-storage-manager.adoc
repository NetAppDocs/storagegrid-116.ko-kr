= SANtricity 스토리지 관리자를 구성합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
SANtricity 스토리지 관리자를 사용하여 StorageGRID 어플라이언스에서 스토리지 디스크 및 하드웨어 구성 요소의 상태를 모니터링할 수 있습니다. 이 소프트웨어에 액세스하려면 E2700 컨트롤러(어플라이언스의 스토리지 컨트롤러)의 관리 포트 1의 IP 주소를 알아야 합니다.
