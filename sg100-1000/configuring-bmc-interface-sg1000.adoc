= BMC 인터페이스 구성(SG100 및 SG1000)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
서비스 어플라이언스의 BMC(베이스보드 관리 컨트롤러)에 대한 사용자 인터페이스는 하드웨어에 대한 상태 정보를 제공하고 서비스 어플라이언스에 대한 SNMP 설정 및 기타 옵션을 구성할 수 있도록 합니다.
