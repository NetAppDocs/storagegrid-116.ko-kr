= 어플라이언스 노드 클론 복제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID에서 어플라이언스 노드를 클론 복제하여 새로운 설계 또는 향상된 기능의 어플라이언스를 사용할 수 있습니다. 클론 복제는 기존 노드의 모든 정보를 새 어플라이언스로 전송하고, 수행하기 쉬운 하드웨어 업그레이드 프로세스를 제공하며, 어플라이언스 교체를 위한 서비스 해제 및 확장 대안을 제공합니다.
