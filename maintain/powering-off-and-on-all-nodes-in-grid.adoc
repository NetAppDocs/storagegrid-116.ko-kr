= 그리드의 모든 노드를 끕니다
:icons: font
:imagesdir: ../media/


[role="lead"]
예를 들어 데이터 센터를 이동하는 경우 전체 StorageGRID 시스템을 종료해야 할 수 있습니다. 다음 단계에서는 제어 방식의 셧다운 및 시작을 수행하는 데 권장되는 순서에 대해 개략적으로 설명합니다.

사이트 또는 그리드의 모든 노드의 전원을 끌 경우 스토리지 노드가 오프라인 상태인 동안에는 수집된 개체에 액세스할 수 없습니다.
