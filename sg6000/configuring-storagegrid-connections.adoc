= StorageGRID 연결 구성(SG6000)
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID 시스템에서 StorageGRID 어플라이언스를 스토리지 노드로 구축하려면 먼저 어플라이언스와 사용할 네트워크 간의 연결을 구성해야 합니다. SG6000-CN 컨트롤러(컴퓨팅 컨트롤러)에 사전 설치된 StorageGRID 어플라이언스 설치 프로그램으로 이동하여 네트워킹을 구성할 수 있습니다.
