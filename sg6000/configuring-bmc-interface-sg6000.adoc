= BMC 인터페이스(SG6000) 구성
:icons: font
:imagesdir: ../media/


[role="lead"]
SG6000-CN 컨트롤러의 BMC(베이스보드 관리 컨트롤러)에 대한 사용자 인터페이스는 하드웨어에 대한 상태 정보를 제공하며 SG6000-CN 컨트롤러에 대한 SNMP 설정 및 기타 옵션을 구성할 수 있습니다.
